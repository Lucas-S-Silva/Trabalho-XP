# 🍽️ Cardápio CRUD

## 📌 Sobre o projeto

Este é um sistema simples de cardápio desenvolvido para fins acadêmicos.
O projeto permite cadastrar, visualizar, editar e remover produtos diretamente na tela.

Os dados são armazenados apenas em memória, sem uso de banco de dados.

---

## 🚀 Funcionalidades

* ➕ Adicionar produtos
* 📋 Listar produtos em cards
* ✏️ Editar produtos
* ❌ Remover produtos
* 💰 Exibir valor
* 🖼️ Selecionar imagem do produto

---

## 🛠️ Tecnologias

* HTML
* CSS
* JavaScript
* Bootstrap

## ▶️ Como executar

1. Clone o repositório:

```
git clone https://github.com/Lucas-S-Silva/Trabalho-XP.git
```

2. Acesse a pasta do projeto

3. Abra o arquivo `index.html` no navegador
